Optimistic Oracle

The Optimistic Oracle is used to retrieve prices or other data and put them on-chain. It has an inbuilt resolution system in case of a dispute.

Who is it for?

DeFi projects whose users demand the highest level of financial security and recourse against exploits and errors.

L2 markets to ensure accurate resolution with very few calls to L1.

Prediction markets, options platforms, and other market types that need precise information to resolve.

How is it better?

The OO has an inbuilt dispute resolution system, so any attempt to manipulate the oracle would trigger a dispute.

For L2 solutions, a cross-chain bridge can ensure efficient settlement and security for L2 products.

The OO resolves quickly in the vast majority of cases, and does so in a gas efficient manner.
